# 资源文件说明

## TabBar 图标

为了完整的界面效果，需要在 `tabbar` 目录下添加以下图标文件：

### 必需的图标文件：

```
assets/tabbar/
├── home.png          # 首页图标 (未选中状态)
├── home-active.png   # 首页图标 (选中状态)
├── admin.png         # 管理图标 (未选中状态)
├── admin-active.png  # 管理图标 (选中状态)
├── user.png          # 我的图标 (未选中状态)
└── user-active.png   # 我的图标 (选中状态)
```

### 图标规格要求：

- 尺寸：81px × 81px
- 格式：PNG
- 背景：透明
- 颜色：未选中状态建议使用 #999999，选中状态使用 #0052d9

### 设计建议：

- 图标风格应保持一致
- 线条粗细适中，确保在小尺寸下清晰可见
- 可以使用 Iconfont 或设计师提供的图标
- 建议使用简洁的线条图标风格

## 其他资源

- Logo 图片
- 占位图片等
