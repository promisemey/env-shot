# 问题上报界面修复总结

## 修复的问题

### 1. ✅ 社区选择器界面优化

- **问题**: 选择区域布局不够美观，缺少视觉引导
- **解决**:
  - 调整了 picker 的高度为 `min-height: 88rpx`
  - 增加了垂直 padding: `20rpx 24rpx`
  - 添加了位置图标作为视觉引导
  - 优化了激活状态的视觉反馈

### 2. ✅ 问题分类选中样式重新设计

- **问题**: 选中后图标和文字变成白色，在白色背景下看不见
- **解决**:
  - 改变选中状态的背景色为 `var(--td-brand-color-1)` (浅蓝色)
  - 图标颜色始终保持 `var(--td-brand-color)` (深蓝色)
  - 添加了右上角的选中标记 (✓)
  - 文字颜色保持深色，确保可读性

### 3. ✅ 图标兼容性修复

- **问题**: 部分图标在 TDesign 中不存在
- **解决**: 替换了不存在的图标：
  - `drop` → `circle` (水质问题)
  - `weather` → `cloud` (空气质量)
  - `flower` → `tree` (绿化问题)
